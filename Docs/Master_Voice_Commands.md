# VoiceBook Studio — Master Voice Commands Reference

**Version 1.0**

---

## How Voice Commands Work

VoiceBook Studio has two voice input paths:

- **Built-in microphone** — active when Dragon and JSay are not detected. Listens continuously using Windows Speech Recognition. Say any command below at any time.
- **Dragon / JSay** — Dragon dictates into the editor naturally. Voice commands work best when focus is in the Chat input (Panel 3). See the AT Setup Guide for how to map Dragon and JSay commands to keyboard shortcuts.
- **Chat input shortcut** — type any command below into the chat input box and press Enter. This works with any input method including on-screen keyboard.

Commands are not case-sensitive.
