# VoiceBook Studio

A writing app for users of Dragon, JSay, and JAWS programs.

## Overview

VoiceBook Studio is a WPF desktop application designed to assist users of voice control and screen reader software in writing and producing audiobooks and structured documents.

## Features

- Voice command integration (Dragon, JSay, JAWS)
- Chapter detection and management
- AI-assisted writing (Claude API)
- Azure Text-to-Speech support
- Export to DOCX and PDF
- Prompt library for reusable writing prompts
- Project management with save/load support

## Getting Started

1. Open `VoiceBookStudio.sln` in Visual Studio
2. Restore NuGet packages
3. Build and run the project
4. Configure your API keys via the settings dialog on first launch

## Requirements

- Windows 10 or later
- .NET 8.0 (WPF)
- Visual Studio 2022 recommended
